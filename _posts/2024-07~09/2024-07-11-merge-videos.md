  使用 ffmpeg 合并视频

# 使用 ffmpeg

* 方法1: 本地安装
```sh
$ brew install ffmpeg
```

* 方法2:使用 wasm

```sh
$ open https://ffmpegwasm.netlify.app/playground/
```

# 合并

本地仅需一条命令即可：

```sh
$ ffmpeg -i left.mp4 -i right.mp4 -filter_complex hstack output.mp4
```

或直接使用在线工具执行上述命令
