  lldb 及目前常用的小技巧


## lldb

```sh
# main 断点
(lldb) b main

# 根据地址设置断点
(lldb) b 0x001

# 启动断点
(lldb) process launch --stop-at-entry

# 下一步
(lldb) n
# 回车下一步，回车是重复执行上一个命令
(lldb) [ENTER]

# 读取所有寄存器
(lldb) re r -a

# 读取内存
(lldb) x 0x001
```
