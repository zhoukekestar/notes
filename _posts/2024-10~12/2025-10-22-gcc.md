


  GCC (GNU Compiler Collection)

# Install

* gcc

```sh
$ brew install gcc
```

* [binutils](https://en.wikipedia.org/wiki/GNU_Binutils)，包含常见的
  * as 汇编
  * nm 列出导出符号
  * ld 连接
  * objdump 转储目标文件信息

```sh
$ brew install binutils
```
